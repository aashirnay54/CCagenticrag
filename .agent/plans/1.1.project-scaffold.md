# 1.1 Project Scaffold

✅ **Simple** - Single-pass executable, low risk

## Context
Starting from an empty repo (docs only). This sub-plan creates the full directory skeleton for both the React frontend and FastAPI backend, installs all dependencies, and verifies both dev servers run. Every other sub-plan depends on this one.

## Tasks

### Frontend
1. Bootstrap Vite project at `frontend/` using `npm create vite@latest frontend -- --template react-ts`
2. Install and configure Tailwind CSS v3: `npm install -D tailwindcss postcss autoprefixer && npx tailwindcss init -p`
   - Add `tailwind.config.ts` with `content: ["./index.html", "./src/**/*.{ts,tsx}"]`
   - Add base directives to `frontend/src/index.css`: `@tailwind base; @tailwind components; @tailwind utilities;`
3. Install and configure shadcn/ui: `npx shadcn-ui@latest init` (select neutral base color, CSS variables enabled, TypeScript yes)
4. Install core frontend deps:
   ```
   npm install @supabase/supabase-js @supabase/auth-helpers-react react-router-dom lucide-react
   ```
5. Create `frontend/.env.example`:
   ```
   VITE_SUPABASE_URL=
   VITE_SUPABASE_ANON_KEY=
   VITE_API_URL=http://localhost:8000
   ```
6. Create `frontend/src/lib/supabase.ts` — Supabase client singleton:
   ```ts
   import { createClient } from '@supabase/supabase-js'
   export const supabase = createClient(
     import.meta.env.VITE_SUPABASE_URL,
     import.meta.env.VITE_SUPABASE_ANON_KEY
   )
   ```

### Backend
7. Create `backend/` directory with Python venv: `python3 -m venv backend/venv`
8. Create `backend/requirements.txt`:
   ```
   fastapi
   uvicorn[standard]
   python-dotenv
   supabase
   openai
   langsmith
   pydantic
   pydantic-settings
   httpx
   python-multipart
   ```
9. Create `backend/main.py` — FastAPI app with CORS and health endpoint:
   ```python
   from fastapi import FastAPI
   from fastapi.middleware.cors import CORSMiddleware

   app = FastAPI()

   app.add_middleware(
       CORSMiddleware,
       allow_origins=["http://localhost:5173"],
       allow_credentials=True,
       allow_methods=["*"],
       allow_headers=["*"],
   )

   @app.get("/health")
   def health():
       return {"status": "ok"}
   ```
10. Create `backend/.env.example`:
    ```
    SUPABASE_URL=
    SUPABASE_SERVICE_ROLE_KEY=
    OPENAI_API_KEY=
    LANGSMITH_API_KEY=
    LANGSMITH_PROJECT=
    LANGSMITH_TRACING=true
    ```

### Root
11. Update root `.gitignore`:
    ```
    node_modules/
    frontend/dist/
    backend/venv/
    **/.env
    __pycache__/
    *.pyc
    .DS_Store
    ```

## Validation
- `cd frontend && npm run dev` → browser opens at `http://localhost:5173`, default Vite React page renders without console errors
- `cd backend && source venv/bin/activate && pip install -r requirements.txt && uvicorn main:app --reload` → `curl http://localhost:8000/health` returns `{"status":"ok"}`

## Dependencies
None — this is the first sub-plan.
