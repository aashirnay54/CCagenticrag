# 1.3 Auth UI

⚠️ **Medium** - May need iteration, some complexity

## Context
Implements login/signup pages using Supabase Auth and a protected route wrapper. After this sub-plan, unauthenticated users hitting any route are redirected to `/login`; authenticated users land on `/chat`. The JWT from Supabase Auth will later be passed as `Authorization: Bearer <token>` on every backend API request.

## Tasks

### shadcn/ui Components
1. Add required components: `npx shadcn-ui@latest add button input label card`

### Auth Context
2. Create `frontend/src/contexts/AuthContext.tsx`:
   ```tsx
   import { createContext, useContext, useEffect, useState } from 'react'
   import { Session, User } from '@supabase/supabase-js'
   import { supabase } from '../lib/supabase'

   interface AuthContextType {
     user: User | null
     session: Session | null
     loading: boolean
     signOut: () => Promise<void>
   }

   const AuthContext = createContext<AuthContextType | undefined>(undefined)

   export function AuthProvider({ children }: { children: React.ReactNode }) {
     const [user, setUser] = useState<User | null>(null)
     const [session, setSession] = useState<Session | null>(null)
     const [loading, setLoading] = useState(true)

     useEffect(() => {
       supabase.auth.getSession().then(({ data: { session } }) => {
         setSession(session)
         setUser(session?.user ?? null)
         setLoading(false)
       })
       const { data: { subscription } } = supabase.auth.onAuthStateChange((_event, session) => {
         setSession(session)
         setUser(session?.user ?? null)
       })
       return () => subscription.unsubscribe()
     }, [])

     const signOut = async () => { await supabase.auth.signOut() }

     return (
       <AuthContext.Provider value={{ user, session, loading, signOut }}>
         {children}
       </AuthContext.Provider>
     )
   }

   export const useAuth = () => {
     const ctx = useContext(AuthContext)
     if (!ctx) throw new Error('useAuth must be used within AuthProvider')
     return ctx
   }
   ```

### Entry Point
3. Update `frontend/src/main.tsx` to wrap app with `<AuthProvider>` and `<BrowserRouter>`:
   ```tsx
   import { BrowserRouter } from 'react-router-dom'
   import { AuthProvider } from './contexts/AuthContext'
   // ...
   root.render(
     <BrowserRouter>
       <AuthProvider>
         <App />
       </AuthProvider>
     </BrowserRouter>
   )
   ```

### Protected Route
4. Create `frontend/src/components/ProtectedRoute.tsx`:
   ```tsx
   import { Navigate, Outlet } from 'react-router-dom'
   import { useAuth } from '../contexts/AuthContext'

   export function ProtectedRoute() {
     const { user, loading } = useAuth()
     if (loading) return <div className="flex h-screen items-center justify-center">Loading...</div>
     if (!user) return <Navigate to="/login" replace />
     return <Outlet />
   }
   ```

### Pages
5. Create `frontend/src/pages/LoginPage.tsx` — email + password form:
   - Uses shadcn/ui `Card`, `Input`, `Button`, `Label`
   - On submit: `supabase.auth.signInWithPassword({ email, password })`
   - On success: `navigate('/chat')`
   - On error: display error message below form
   - Footer link: "Don't have an account? Sign up" → `/signup`

6. Create `frontend/src/pages/SignupPage.tsx` — email + password + confirm password form:
   - On submit: validate passwords match, then `supabase.auth.signUp({ email, password })`
   - On success: show "Check your email for a confirmation link" message
   - If email confirm is disabled in Supabase, redirect to `/chat` on success
   - Footer link: "Already have an account? Log in" → `/login`

### Router
7. Replace `frontend/src/App.tsx` with React Router routes:
   ```tsx
   import { Routes, Route, Navigate } from 'react-router-dom'
   import { ProtectedRoute } from './components/ProtectedRoute'
   import { LoginPage } from './pages/LoginPage'
   import { SignupPage } from './pages/SignupPage'

   function ChatPlaceholder() {
     const { signOut } = useAuth()
     return (
       <div className="p-4">
         <p>Chat coming soon</p>
         <button onClick={signOut}>Log out</button>
       </div>
     )
   }

   export default function App() {
     return (
       <Routes>
         <Route path="/login" element={<LoginPage />} />
         <Route path="/signup" element={<SignupPage />} />
         <Route element={<ProtectedRoute />}>
           <Route path="/chat" element={<ChatPlaceholder />} />
           <Route path="/" element={<Navigate to="/chat" replace />} />
         </Route>
       </Routes>
     )
   }
   ```

## Validation
- Visit `http://localhost:5173/` unauthenticated → redirected to `/login`
- Visit `http://localhost:5173/chat` unauthenticated → redirected to `/login`
- Sign up with a test email → either email confirmation or immediate login depending on Supabase settings
- Log in with valid credentials → lands on `/chat` placeholder
- Log out → back to `/login`
- Refresh on `/chat` while logged in → stays on `/chat` (session persisted)
